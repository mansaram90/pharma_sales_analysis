# Pharma Sales Data Analysis

This project analyzes pharmaceutical sales data to uncover trends in revenue, marketing effectiveness, and inventory balance.

##  Contents
- `Pharma sales data analysis.ipynb`: main analysis notebook
- Data includes: sales orders, marketing spend, inventory, shipments, products
- pharma sales data.zip
- Pharma_Sales_Data_Analysis_Report.docx

##  Key Insights
- Top-performing territories and products
- Correlation between marketing spend and sales
- Inventory vs shipment balance
- Sales trends over time

## Technologies
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook
